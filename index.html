This audio game is intended for play without visual feedback.

Try creating a map of each level in your mind.
Help your partner to gather all outputs that belong to the same melody.

The game uses a total of 4 buttons.
The main 3 buttons are: "Move", "Pick" and "Drop".
The "Move" choice gives 4 options:
"Go forward";
"Go left";
"Go right";
"Go back";

There are 3 input methods:
The first is by using the arrow keys on the keyboard.
The second is by clicking on the top, bottom left or right side of the screen.
The third input method uses the numbers on the keyboard from 1 to 4.

Press anything to start.
